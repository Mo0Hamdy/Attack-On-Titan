# 🛡️ Attack on Titan Landing Page

A responsive landing page inspired by the popular anime **Attack on Titan**. The project showcases the main characters, seasons, and a brief overview using a clean and modern UI built with HTML and CSS.

## ✨ Features

- 📱 Responsive design
- 🎨 Modern and clean user interface
- 🦸 Character cards with images and descriptions
- 📺 Seasons section with short summaries
- 📖 About section
- 📞 Contact section with social media icons
- ⚡ Smooth scrolling navigation
- 🎭 Hover effects and subtle animations

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Font Awesome
- Google Fonts (Work Sans)
